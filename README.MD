# BOLT-util
BOLT-util adalah CommandLine multitool untuk unlock BOLT Orion Movimax

## Instalasi
```bash
npm install
```
## Fitur
- #### ganti username dan password
- #### unlock versi 17, 19, dan 22
- #### IMEI generator
## Cara Penggunaan
```bash
node .
```
## Menggunakan .bat
```bat
START "BOLT UTIL" /D "[INSTALL DIR]" "node" "."
```
